Exercice 1: Affichage des premières lignes d'un jeu de données

Dans cet exercice, vous devez afficher les premières lignes du jeu de données `iris`.

# Charger le jeu de données iris # Afficher les cinq premières lignes du jeu de données # Charger le jeu de données iris data(iris) # Afficher les cinq premières lignes du jeu de données head(iris) ex() %>% check_function("head", args = "x") %>% check_equal()
Pour afficher les premières lignes d'un jeu de données en R, utilisez la fonction head().
